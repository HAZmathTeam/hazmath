Install additional packages required by the haznics demos: 
- edit "install_haznics.sh" to match your system
- in terminal, run ./install_haznics.sh

# Running tests:
```
source setup.rc
cd tests/
python3 test_haznics.py
```

# Running demos, e.g. Poisson equation:
```
source setup.rc
python3 poisson.py
```
